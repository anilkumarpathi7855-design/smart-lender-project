# Technologies Used
## Description
The project was developed using a set of powerful Python-based tools and libraries for data processing, machine learning, visualization, and deployment. These technologies provide the foundation for efficient model development, analysis, and application deployment.

## Anaconda Navigator

Anaconda Navigator is a desktop graphical user interface used to launch applications and manage Python packages and environments for data science.

Download:
https://www.anaconda.com/download

---

## PyCharm

PyCharm is an Integrated Development Environment (IDE) specifically designed for Python programming, offering code analysis and debugging tools.

Official Website:
https://www.jetbrains.com/pycharm/

---

## NumPy

NumPy is the core library for numerical computing in Python, providing support for large, multi-dimensional arrays and mathematical functions.

Documentation:
https://numpy.org/doc/stable/

---

## Pandas

Pandas is a powerful data manipulation and analysis library offering data structures and operations for handling numerical tables and time series.

Documentation:
https://pandas.pydata.org/docs/

---

## Scikit-learn

Scikit-learn is a machine learning library featuring classification, regression, and clustering algorithms such as Support Vector Machines, Decision Trees, Random Forests, and Logistic Regression.

Documentation:
https://scikit-learn.org/stable/

---

## Matplotlib

Matplotlib is a comprehensive library for creating static, animated, and interactive visualizations in Python.

Documentation:
https://matplotlib.org/stable/

---

## Seaborn

Seaborn is a statistical data visualization library based on Matplotlib that provides a high-level interface for creating attractive and informative graphics.

Documentation:
https://seaborn.pydata.org/

---

## Flask

Flask is a lightweight Python web framework used to develop web applications and deploy machine learning models.

Documentation:
https://flask.palletsprojects.com/

---

## Conclusion

The Smart Lender – Loan Approval Prediction project uses Python along with powerful libraries such as NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn, and Flask. These technologies help perform data preprocessing, machine learning model development, visualization, and deployment efficiently.
